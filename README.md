# Labbaik HR · BPKH Employee Self-Service

Modern HRIS for Badan Pengelola Keuangan Haji with Islamic-aware features.

## Features
- Face recognition attendance (face-api.js)
- GPS geofence (PostGIS)
- Audit trail ISO 27001 (hash-chained)
- Tanya HR AI (Groq LLaMA 3.3 70B)
- Syariah-aware (prayer times, Hijri calendar, cuti haji/umrah)

## Stack
- Frontend: React 19 + Vite 6
- Backend: FastAPI + Supabase
- AI: Groq LLaMA 3.3 70B
- Face: face-api.js (128-dim embeddings)
